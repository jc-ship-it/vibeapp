MVP 功能与范围
=============

目标
----
在 iOS 原生端实现截图复盘的最小可用闭环：选截图 → 本地 OCR → 结构化卡片 → 时间线浏览 → 云端文本分析（摘要/关联/趋势）。

功能清单
--------
1. 截图导入
   - 从相册选择多张截图（PHPicker）。
   - 支持标注来源 App（如小红书、B 站等）。
   - 允许重复选择并去重（同一图片哈希判定）。

2. 本地 OCR
   - 使用 Apple Vision 提取文字。
   - 记录语言、置信度与分段文本。

3. 结构化卡片
   - 字段：时间、来源 App/场景、OCR 原文、标签、摘要、关联关键词。
   - 支持用户手动编辑与补充。

4. 时间线复盘
   - 按时间顺序展示卡片。
   - 基础筛选（关键词、标签、日期区间）。

5. 云端文本分析
   - 仅上传 OCR 文本与必要元数据。
   - 返回摘要、相似点、兴趣趋势。

非目标（MVP 之外）
-----------------
- 不做复杂的社交与分享功能。
- 不强制上传图片；默认只上传文字。
- 不做复杂的多语言自动翻译。

成功标准
--------
- 用户可以从多张截图中得到可读的时间线。
- 关键内容可被搜索与筛选。
- AI 输出能帮助用户快速复盘。
