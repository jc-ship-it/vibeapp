VibeApp
=======

一个 iOS 截图复盘应用，帮助用户从相册截图中提取文字、整理信息，并按时间线复盘与总结。

目标
----
- iOS 原生首发（Swift/SwiftUI）
- 截图本地 OCR，云端仅做文本分析与关联
- 支持时间线回顾与趋势洞察

目录
----
- docs/MVP.md：MVP 功能与范围
- docs/UserFlow.md：核心用户流程
- docs/DataModel.md：本地数据结构
- docs/Privacy.md：隐私与权限策略

里程碑
------
- 阶段 1：iOS 骨架 + 图片选择 + OCR
- 阶段 2：结构化整理 + 时间线
- 阶段 3：云端分析接口 + 客户端对接
- 阶段 4：登录与同步
- 阶段 5：上架准备
